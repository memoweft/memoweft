对话模型 必填

整理记忆的小模型 可选

在后台把你说的话整理成记忆时用它，通常更快更省。不填就用上面那个对话模型。

导出我的记忆包

把它记住你的全部内容（记忆线索、经历片段、对你的理解，以及它们之间的关联）打包成一个文件存到本地。可以拿去备份，或换个地方接着用。你的密钥、聊天记录不在里面。

会下载一个 .bundle.json 文件到你的电脑。

从记忆包恢复

选一个之前导出的 .bundle.json 文件，把里面的记忆合并进来。已经有的不会重复导入。导入前会先给你看一遍将要导入多少、跳过多少，确认后才真正写入。

清空全部记忆 · 重新开始

把它记住你的一切都清空——所有记忆线索、经历片段、对你的理解，以及这些操作的记录，全部抹掉，回到刚认识你时的空白。当前的对话会被收起（不删）、并开一段新的空对话。此操作不可恢复。建议先"导出我的记忆包"留个备份。
