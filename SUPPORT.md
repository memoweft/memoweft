# Support

## Usage questions

Use [GitHub Discussions](https://github.com/memoweft/memoweft/discussions) for installation help, integration questions, architecture discussions, and examples.

Before posting, include:

- the MemoWeft version;
- Node and operating-system versions;
- the SQLite driver and integration in use;
- a minimal, sanitized example;
- expected and actual behavior.

## Bugs and feature requests

Use the [issue tracker](https://github.com/memoweft/memoweft/issues) for reproducible bugs and concrete feature requests. The issue templates collect the information needed to investigate efficiently.

## Security

Do not report vulnerabilities publicly. Follow the private process in [SECURITY.md](./.github/SECURITY.md).

## Scope

Community support is provided for the current npm release and the current `main` branch. The reference host demonstrates integration patterns; production deployment, authentication, consent UI, and infrastructure remain the responsibility of the host application.
